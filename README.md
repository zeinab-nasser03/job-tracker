# 💼 Job Application Tracker

A modern and responsive Job Application Tracker built with React and Vite. This application helps users organize and monitor their job and internship applications by tracking company names, positions, and application statuses.

---

## 🚀 Features

- ✅ Add new job applications
- ✅ Delete applications
- ✅ Filter applications by status
- ✅ Track application progress
- ✅ Dashboard statistics
- ✅ Local Storage persistence
- ✅ Responsive design for desktop and mobile
- ✅ Built with React components and hooks

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/job-application-tracker.git
```

Navigate to the project directory:

```bash
cd job-application-tracker
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

---

## 📋 Usage

1. Enter a company name.
2. Enter the position title.
3. Select the application status.
4. Click **Add Job**.
5. View statistics and manage your applications.
6. Filter applications by status.
7. Delete applications when needed.
8. Data remains saved after page refresh thanks to Local Storage.
